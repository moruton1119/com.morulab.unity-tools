# BLM Connector

BOOTH Library Manager(BLM)에서 관리하는 에셋을 Unity 내에서 쇼핑몰처럼 탐색하고 가져올 수 있는 도구입니다.

---

## 기본 사용법

1. **Morulab Launcher** 또는 **Morulab > BLM Connector**에서 열기
2. 그리드에서 에셋을 클릭하여 상세 정보 확인
3. 필요한 파일을 선택하여 가져오기

---

## 필터

사이드바 드롭다운으로 표시할 에셋을 필터링할 수 있습니다.

| 옵션 | 내용 |
|------|------|
| All Products | 모든 에셋 |
| BLM Products | BLM 관리 에셋만 |
| Local Products | LocalAssets 폴더 에셋만 |
| 📋 [목록 이름] | BLM에서 만든 커스텀 목록 |

---

## 가져오기 방법

### 방법 1: 더블클릭 (빠른 가져오기)
제품 카드를 더블클릭하면 해당 제품의 모든 패키지를 한 번에 가져옵니다.

### 방법 2: 개별 선택
1. 제품을 클릭하여 모달 열기
2. 각 파일의 "Import" 버튼으로 필요한 것만 가져오기
3. "Select All / Deselect"로 일괄 선택 가능

### 방법 3: 대기열 시스템
1. 여러 제품을 "Add to Queue"로 대기열에 추가
2. "Process Queue"로 일괄 실행
3. "Show Import Dialog" ON/OFF로 대화상자 표시 제어

---

## 가져오기 상태

그리드의 제품 카드에서 가져오기 상태를 확인할 수 있습니다.

| 상태 | 표시 | 설명 |
|------|------|------|
| 미가져오기 | 일반 표시 | 아직 가져오지 않음 |
| 설치됨 | 흐리게 표시 | 이전에 가져오기 완료 |
| 이번 세션 가져오기 | 파란 테두리 | 이번 세션에서 완료 |

---

## 가져온 에셋 관리

가져오기가 완료된 제품의 모달 하단에 다음 버튼이 표시됩니다.

| 버튼 | 기능 |
|------|------|
| Show in Project | 가져온 폴더를 Project 창에서 포커스 |
| Delete | 가져온 폴더 삭제 (확인 대화상자 있음) |

---

## LocalAssets 기능

BLM에서 관리하지 않는 에셋도 같은 UI로 관리할 수 있습니다.

### 사용 방법
1. 사이드바의 "Open LocalAssets Folder" 클릭
2. LocalAssets 폴더 내에 에셋용 폴더 생성
3. 폴더 이름이 제품 이름으로 표시됩니다
4. `thumbnail.png`를 넣으면 썸네일이 표시됩니다

---

## 문제 해결

| 문제 | 해결 방법 |
|------|----------|
| 에셋이 표시되지 않음 | "Refresh DB" 클릭 |
| 필터가 작동하지 않음 | "Refresh DB" 클릭 |
| Show in Project가 없음 | 먼저 에셋을 가져오세요 |
