# BLM Connector (Booth Library Manager)

로컬 BOOTH Library Manager (BLM) 데이터베이스에서 직접 에셋을 관리하고 가져옵니다.

## 주요 기능

- **BLM 동기화**: BLM의 `data.db`를 읽어 구매한 에셋 목록을 표시합니다.
- **썸네일 그리드**: 시각적인 썸네일로 에셋을 탐색할 수 있습니다.
- **선택적 가져오기**: 제품 폴더 내에서 특정 `.unitypackage` 파일만 선택하여 가져올 수 있습니다.
- **가져오기 대기열**: 여러 패키지를 순차적으로 안전하게 처리하여 Unity 충돌을 방지합니다.
- **설치 상태 추적**: 제품이 이미 성공적으로 가져오기 되었는지 자동으로 감지합니다.

## 사용 방법

1. **Morulab Launcher**에서 이 도구를 엽니다.
2. 그리드에서 에셋을 클릭하여 상세 정보를 확인합니다.
3. 상세 패널에서 가져오려는 패키지를 선택합니다.
4. **대기열에 추가(Add to Queue)** 후 **대기열 처리(Process Queue)**를 클릭하여 가져오기를 시작합니다.

## 권장 사항

SQLite 라이브러리가 필요합니다. 누락된 경우 도구에서 설치 안내 또는 내장된 폴더를 사용하도록 안내합니다.
